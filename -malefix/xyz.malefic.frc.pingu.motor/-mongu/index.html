Mongu

class Mongu<T : Any>(val motor: T, val engu: Engu? = null, enguConfig: CANcoderConfiguration.() -> Unit = {}, monguConfig: MonguConfig<out T>.() -> Unit = {})

A generic motor wrapper class that allows configuration of different motor types.

Make sure to configure the motor after instantiation to apply desired settings.

Parameters

T

The type of motor being wrapped.

monguConfig

A lambda that applies configuration settings to the motor.
